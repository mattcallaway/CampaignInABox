## Critical User Flows

### Flow 1 — First-Time Setup
```
1. Create campaign       → Admin → Campaign Admin → Create
2. Upload results file   → Data → Data Manager → Upload New File
3. Run pipeline          → System → Pipeline Runner → Run
4. View map              → Geography → Precinct Map
5. View archive          → Intelligence → Historical Archive
```

### Flow 2 — Re-Running After Data Update
```
1. Upload new file       → Data Manager → Upload New File
2. Verify file registered → Data Manager → File Registry
3. Re-run pipeline       → Pipeline Runner → select contest → Run
4. Refresh map page      → browser refresh is sufficient
```

### Flow 3 — Diagnosing a Problem
```
1. Check diagnostics     → Diagnostics page
2. Download pipeline log → Pipeline Runner → Download Log
3. Check precinct IDs    → Data Manager → Precinct ID Review
4. Review crosswalk      → reports/crosswalk_repair/
```
